## 🛠️ Detailed Task Breakdowns

### 🔹 Task 1: Data Immersion & Wrangling
Focused on cleaning and restructuring messy raw sales transaction files into an analysis-ready tabular dataset.
* **Key Steps**: Formulated a comprehensive data dictionary, built a reproducible Python/Pandas cleaning script to standardize date-times, resolved null values, eliminated duplicate rows, handled outlier fields, and recalculated transaction totals.
* **Highlights**: [clean_sales_data.py](https://github.com/luckyghanghas/Task-1-Data-Immersion-Wrangling/blob/main/scripts/clean_sales_data.py)

### 🔹 Task 2: Exploratory Data Analysis (EDA) & SQL
Focused on extracting early insights and identifying core trends in sales performance.
* **Key Steps**: Formulated business questions on sales channels, customer age cohorts, and campaign ROI. Wrote SQL queries and generated aggregate summaries. Created a static spreadsheet layout outlining early dashboard design patterns.
* **Highlights**: [sales_queries.sql](https://github.com/luckyghanghas/Task-2-EDA/blob/main/scripts/sales_queries.sql)

### 🔹 Task 3: Interactive Dashboarding (Cohort, Funnel & Segment)
Focused on implementing structured analytical pipelines to track user retention and conversion.
* **Key Steps**: Formulated interactive dashboards centering around cohort analysis, purchase funnels, and demographic segments. Defined KPIs mathematically in the documentation with formulas and business rationale.
* **Highlights**: [app.py](https://github.com/luckyghanghas/Task-3-Dashboard/blob/main/dashboard/app.py)

### 🔹 Task 4: Data Storytelling & Statistical Validation
Focused on validating business claims through rigorous statistics and compiling an executive business proposal.
* **Key Steps**: Built a Python testing script using `scipy.stats` to execute Welch's T-Test, Two-Proportion Z-Test, and Chi-Square tests comparing customer satisfaction across sales channels. Generated a dynamic PowerPoint deck.
* **Highlights**: [hypothesis_testing.py](https://github.com/luckyghanghas/Task-4-Storytelling/blob/main/scripts/hypothesis_testing.py) and [Task 4 Presentation](https://github.com/luckyghanghas/Lucky-DataAnalyst-Internship-Portfolio/blob/main/presentation/data_storytelling_and_statistical_validation.pptx)

## Completed Work

### Task 1: Data Immersion & Wrangling

Task 1 focused on preparing raw data for analysis. The work included dataset familiarization, data dictionary creation, data quality assessment, data cleaning, transformation, and creation of an analysis-ready dataset.

Key deliverables:

- Raw sample sales dataset
- Data dictionary
- Python/Pandas cleaning script
- Cleaned analysis-ready dataset
- Data quality report
- LinkedIn video script

Repository: [Task-1-Data-Immersion-Wrangling](https://github.com/luckyghanghas/Task-1-Data-Immersion-Wrangling)

### Task 2: Exploratory Data Analysis & Business Intelligence

Task 2 focused on generating business insights from the cleaned dataset. The work included descriptive statistics, SQL-based business questions, query results, KPI selection, and a static Excel dashboard mock-up.

Key deliverables:

- EDA report
- SQL queries for business questions
- SQL query results
- Static Excel dashboard mock-up
- LinkedIn video script

Repository: [Task-2-EDA](https://github.com/luckyghanghas/Task-2-EDA)
